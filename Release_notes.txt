ISE 5.7/6.0 (ISE Eiffel)

	* The Ace files for ISE Eiffel provided in this package are for
	  Eiffel classic. In order to get Ace files configured for Eiffel
	  for .NET, you will need to set the environment variable
	  $GOBO_DOTNET, and then run the following commands:

	    set GOBO_DOTNET=true
	    cd %GOBO%
	    geant clobber
	    geant install

ISE 5.6 (ISE Eiffel)

	* The Gobo Eiffel package has been configured to work with ISE 5.7/6.0.
	  If you are using ISE 5.6, you will need to set the environment
	  variable $ISE_5_6, and then run the following commands to reconfigure
	  the package with ISE 5.6:

	    set ISE_5_6=true
	    cd %GOBO%
	    geant clobber
	    geant install
