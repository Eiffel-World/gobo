Gobo Eiffel: Downloading Instructions

Gobo Eiffel 1.0 consists of five Eiffel libraries:

  • Gobo Eiffel Kernel Library
  • Gobo Eiffel Structure Library
  • Gobo Eiffel Lexical Library
  • Gobo Eiffel Parse Library
  • Gobo Eiffel Utility Library

and three utilities:

  • Gobo Eiffel Lex, version 1.0b (gelex)
  • Gobo Eiffel Yacc, version 1.0a (geyacc)
  • Gobo Eiffel Preprocessor (gepp)

Acknowledgement

Gelex 1.0 has been written in Eiffel after reverse-engineering the flex utility (version 2.5.2) contributed to Berkeley by Vern Paxson. The C source code of the flex utility can be downloaded from ftp://prep.ai.mit.edu/pub/gnu.

Geyacc 1.0 is an adaptation of GNU Bison (version 1.25), distributed by the Free Software Foundation, so that it can generate Eiffel classes instead of C files. The original version of GNU Bison can be downloaded from ftp://prep.ai.mit.edu/pub/gnu. Geyacc is distributed under the same terms and conditions as GNU Bison (see COPYING file for details). In particular, the output of geyacc as well as the skeleton classes from the Gobo Eiffel Parse Library may be used in non-free programs.
